# CyberTrack - Plateforme de Signalement

## 📋 Description
Plateforme nationale de signalement des contenus illicites, fraudes et abus en ligne au Cameroun.

## 👥 Équipe
- **BANGFOFACK** - Lead Dev, Accueil, Structure
- **EKO** - Base de données, Formulaire
- **TAYOUO** - Confirmation, Impression
- **ZUTEU** - Authentification
- **RICHARDA** - Administration

## 🛠️ Technologies
- ASP.NET MVC 5
- ADO.NET
- SQL Server
- HTML5/CSS3/JavaScript

## 🚀 Installation
1. Cloner le repo
2. Ouvrir la solution dans Visual Studio
3. Configurer la connexion SQL dans Web.config
4. Exécuter les scripts SQL
5. Lancer le projet (F5)

## 🌿 Branches
- `main` - Version stable
- `develop` - Intégration
- `feature/*` - Développement individuel
